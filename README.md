# 3. Key Features (주요 기능)
- **회원가입 및 로그인**:
  - 회원가입 시 사용자 정보와 취향 태그가 DB에 등록됩니다.

- **마이페이지**:
  - 회원 정보를 수정하고 주문목록 및 배송현황을 확인할 수 있습니다.
  - 관심 있는 상품과 게시글을 찜하여 마이페이지에서 쉽게 확인할 수 있습니다.

- **쇼핑 페이지 및 검색**:
  - 카테고리별 분류, 인기 상품 필터링, 사용자 취향 기반 상품 추천 기능을 제공합니다.
  - 검색창을 통해 상품을 검색할 수 있습니다. 

- **상품 상세 및 장바구니**:
  - 상품의 상세 정보, 리뷰, 판매자 문의를 확인할 수 있습니다.
  - 상품 옵션을 선택하여 장바구니에 추가 후 구매할 수 있습니다.

- **커뮤니티 자취상담소**:
  - 자취 상담소 Q&A 게시판을 통해 사용자 간 정보 교환이 가능합니다.

- **커뮤니티 집들이**:
  - 집들이 게시판에서 인테리어를 공유하고, 사진과 글을 업로드할 수 있습니다.

# 4. Project Structure (프로젝트 구조)

```plaintext
project/
├── public/                        # 정적 파일 (이미지, 아이콘 등)
│
├── src/                        
│   ├── app/                       # Next.js App Router (페이지 라우팅)
│   │   ├── (user)/                # 사용자 관련 라우팅 그룹
│   │   │   ├── (account)/         # 계정 관련 페이지 그룹
│   │   │   │   ├── login/         # 로그인 페이지
│   │   │   │   └── register/      # 회원가입 페이지
│   │   │   └── my_page/           # 마이페이지
│   │   ├── cart/                  # 장바구니 페이지
│   │   ├── community/             # 커뮤니티 관련 페이지
│   │   ├── products/              # 상품 상세 페이지
│   │   ├── search/                # 검색 페이지
│   │   ├── service/               # 고객센터 관련 페이지
│   │   ├── shopping/              # 쇼핑 관련 페이지
│   │   ├── globals.css            # 전역 스타일
│   │   ├── layout.tsx             # 전역 레이아웃 컴포넌트
│   │   ├── not-found.tsx          # 404 에러 페이지
│   │   └── page.tsx               # 루트 메인 페이지
│   |
│   ├── components/                # 재사용 가능한 UI 컴포넌트
│   ├── data/                     
│   │   ├── actions/               # 서버 액션 (Next.js Server Actions)
│   │   └── functions/             # 서버 함수 모음
│   ├── hooks/                     # 커스텀 React Hooks
│   ├── types/                     # TypeScript 타입 정의
│   ├── utils/                     # 재사용 유틸리티 함수
│   └── zustand/                   # 전역 상태 관리 (Zustand Store)
│
└── README.md                      # 프로젝트 개요 및 실행 가이드
 ```
